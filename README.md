# Furniture Store Full-Stack Application

A full-stack furniture store web application built with React, TypeScript, Spring Boot, and PostgreSQL.

## Features

- Browse furniture products
- Search, filter, and sort products
- View product details
- Add/remove products from cart
- Persist cart using localStorage
- Checkout form with validation
- Save completed orders to PostgreSQL
- REST API built with Spring Boot

## Tech Stack

### Frontend
- React
- TypeScript
- React Router
- CSS

### Backend
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
